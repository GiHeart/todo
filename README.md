# To Do应用

## 介绍

这个项目是基于flask和bootstrap实现的一个网页todo应用。

也是我写的第一个web应用

数据库使用的是MongoDB





## 主要功能

1. 实现了用户登录、注册
2. 每个用户有不同的主页
3. 每个todo都有创建时间和完成时间
4. 如果状态为已完成，则表格会变色
